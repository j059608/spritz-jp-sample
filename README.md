# spritz-jp-sample
文字を高速に読むために文章の中心を自動で揃える spritz を日本語で動作するように試してみました。

## 要点

日本語を揃えて見せるために以下を行いました。

- 表示に日本語等幅フォントを利用する
- ドットで余白を埋めている箇所をを全角スペースにする
- 文章の長さで切る
